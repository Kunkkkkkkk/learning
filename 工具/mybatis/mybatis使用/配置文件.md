```
    map-underscore-to-camel-case: true
```

驼峰命名法：

可以让数据库的下划线方式和java实体类的驼峰方式的参数一一匹配映射

但是在#{}里面写的是你java里的

```sql
 @Insert("INSERT INTO employee (name, username, password, phone, sex, id_number, create_time, update_time, create_user, update_user)\n" +
            "VALUES (#{name}, #{username}, #{password}, #{phone}, #{sex}, #{idNumber}, #{createTime}, #{updateTime}, #{createUser}, #{updateUser});\n")
    void save(Employee employee);
```

来看看手动映射的，十分麻烦

```xml
 <resultMap id="employeeResultMap" type="com.sky.entity.Employee">
        <!-- id字段 -->
        <id property="idNumber" column="id_number"/>
        <!-- 常规字段 -->
        <result property="name" column="name"/>
        <!-- 其他字段 -->
    </resultMap>
```
